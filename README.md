INNER JOIN

Юуг харуулдаг вэ: Зөвхөн хоёуланд нь байгаа өгөгдлийг.

Жишээ: хүн + захиалга

Хүн захиалга өгсөн бол гарна.

Хүн захиалга өгөөгүй бол гарахгүй.
LEFT JOIN (LEFT OUTER JOIN)

Юуг харуулдаг вэ: Баруун талын хүснэгт таарахгүй байсан ч баруун талын орон хоосон байж болно.

Жишээ: хүн + захиалга

Бүх хүн гарах ёстой.

Хэрвээ хүн захиалга өгөөгүй бол захиалгын баганууд хоосон (NULL).

Түлхүүр үг: “Бүгдийг баруун талын холбоосгүй ч харуулна
 RIGHT JOIN (RIGHT OUTER JOIN)

LEFT JOIN-ын эсрэг: баруун талын хүснэгт бүр гарах ёстой, зүүн тал таарахгүй бол хоосон болно.

Жишээ: хүн + захиалга

Бүх захиалга гарна.

Хэрвээ захиалгын эзэн хүн олддоггүй бол хүний нэр хоосон (NULL).

Түлхүүр үг: “Бүх захиалга гарах ёстой”

 FULL JOIN (FULL OUTER JOIN)

LEFT + RIGHT JOIN нийлүүлсэнтэй адил: Бүх зүүн тал + бүх баруун тал, таараагүй нь хоосон.

Жишээ: хүн + захиалга

Хүн захиалга өгөөгүй ч гарах

Захиалга эзэнгүй ч гарах

Түлхүүр үг: “Бүгдийг харуул, таараагүйг хоосон болго
